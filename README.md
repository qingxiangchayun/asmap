asmap
=====
for developer
方便通过url找到jsp path

### 技术选型
前端：jQuery、bootstrap、requirejs
后端：node、express、mongoose
数据库：mongodb


### 安装运行

下载安装工程 
```
$ git clone git@github.com:qingxiangchayun/asmap.git
$ cd asmap
$ npm install
```

安装启动mongodb
安装：[https://www.mongodb.org/] : https://www.mongodb.org/ "mongodb 官网"
启动：`$ ./mongod --dbpath 'db path'`

运行程序  ` $ npm start `
